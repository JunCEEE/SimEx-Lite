Install pysingfel
~~~~~~~~~~~~~~~~~

Clone the pysingfel repository to a temporary path, e.g., ``/tmp``

.. code-block:: bash

    cd /tmp
    git clone -b simex https://github.com/JunCEEE/pysingfel.git

Install with ``pip``

.. code-block:: bash
    
    cd pysingfel
    pip install ./
